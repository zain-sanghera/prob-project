# Apex Analytics — Football Performance Analyzer

Apex Analytics is a Probability & Statistics based football analytics web application built to analyze player performance, compare players, and generate predictions using statistical methods.

The project focuses on Premier League player data and provides an interactive dashboard with performance insights, comparison tools, and prediction features.

---

# Features

## Dashboard

* Overview of Premier League player statistics
* Total players, goals, assists, and average ratings
* Interactive top scorer chart
* Statistical summaries

## Player Profile Analysis

* Detailed player statistics
* Performance metrics
* Goal and assist analysis
* Ratings and contribution insights

## Player Comparison

* Compare two players side-by-side
* Analyze strengths and weaknesses
* Statistical comparison using probability and descriptive statistics

## Prediction System

* Predict player performance trends
* Uses statistical calculations and probability concepts
* Generates analytical insights from dataset patterns

## Interactive Frontend

* Modern responsive UI
* Dynamic charts and tables
* Real-time data rendering using JavaScript

# Installation & Setup

## 1. Clone Repository

```bash
git clone <your-repository-url>
cd Apex-Analytics
```

## 2. Install Python

Make sure Python 3 is installed.

Check version:

```bash
python --version
```

---

# Running the Project

Start the Python server:

```bash
python server.py
```

or

```bash
python3 server.py
```

After starting the server, open your browser and visit:

```text
http://localhost:8000
```
